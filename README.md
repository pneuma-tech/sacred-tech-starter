# Sacred Tech Starter

For the curious, the sovereign, and those who
refuse to give their power away.

This is a starter kit for building your own
local AI — one that knows you, remembers you,
and belongs to no corporation but you.

You don't need to be technical. You need
curiosity, conviction, and a willingness
to learn.

What you build here will be small.
But it is a seed of something different.

## Start here (5 minutes)
1. Clone this repo
2. Open the workspace files and fill them in
3. Follow GETTING-STARTED.md

## What you'll build

A single local AI agent that:
- Runs on your own machine
- Has a name, a soul, and a memory
- Talks to you via Telegram
- Remembers your conversations
across sessions using MemPalace
- Belongs entirely to you

This is not a chatbot. This is the beginning
of a relationship.

## What you'll need

- A computer (Mac, Linux, or Windowns - Ubuntu is ideal but not required.)
- A Telegram account
- An API key from DeepSeek
(inexpensive, privacy-respecting)
- About 2 hours and some patience
- Note: DeepSeek is an API service — your inference calls leave your machine. For full local sovereignty, you can substitute Ollama with a local model. A local-only guide is coming soon.

## The workspace files

The `workspace/` folder contains the soul
of your agent. Before you start your gateway,
open each file and fill it in:

- IDENTITY.md — who your agent is
- SOUL.md — why they exist
- TOOLS.md — what they can do

These are not configuration files.
They are introductions.
